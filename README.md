# ☁️ 67 Days of Cloud – AWS (KodeKloud)

Welcome to my **100 Days of Cloud Challenge** focused on **Amazon Web Services (AWS)** using KodeKloud! 🚀

This repository documents my daily hands-on learning journey where I explore core AWS services, build real-world setups, and strengthen my cloud fundamentals.

---

## 📌 About the Challenge

The goal of this challenge is to:

* Build a strong foundation in AWS
* Gain hands-on experience with real cloud scenarios
* Stay consistent with daily learning
* Document and share progress publicly

---

## 🛠️ Technologies & Services Covered

* EC2 (Elastic Compute Cloud)
* S3 (Simple Storage Service)
* VPC (Virtual Private Cloud)
* IAM (Identity and Access Management)
* RDS (Relational Database Service)
* Lambda
* CloudWatch
* ECR & ECS
* EKS (Kubernetes)
* DynamoDB
* SNS & SQS
* KMS (Key Management Service)
* Load Balancers
* Auto Scaling
